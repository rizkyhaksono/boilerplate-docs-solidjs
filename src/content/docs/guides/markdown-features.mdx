# Markdown Features

Natee Docs supports all standard Markdown features plus GitHub Flavored Markdown (GFM) extensions and custom MDX components.

## Basic Formatting

You can use all the standard Markdown formatting:

- **Bold text** with `**double asterisks**`
- _Italic text_ with `*single asterisks*`
- ~~Strikethrough~~ with `~~tildes~~`
- `Inline code` with backticks

## Headings

Use `#` for headings. Each heading automatically gets an anchor link for easy sharing:

```markdown
# Heading 1

## Heading 2

### Heading 3

#### Heading 4
```

## Code Blocks

Code blocks support syntax highlighting powered by Shiki with dual-theme support:

```typescript
// TypeScript example with syntax highlighting
interface DocConfig {
  title: string
  description: string
  sidebar: SidebarCategory[]
}

function getDocBySlug(slug: string): DocConfig | null {
  const docs = getAllDocs()
  return docs.find((doc) => doc.slug === slug) ?? null
}
```

```jsx
// JSX component example
function Button(props) {
  return (
    <button class="btn btn-primary" onClick={props.onClick}>
      {props.children}
    </button>
  )
}
```

```css
/* CSS example */
.prose h2 {
  font-size: 1.5rem;
  font-weight: 600;
  border-bottom: 1px solid var(--border-color);
  padding-bottom: 0.5rem;
}
```

## Tables

GFM tables are fully supported:

| Syntax        | Description | Example                     |
| ------------- | ----------- | --------------------------- |
| `**bold**`    | Bold text   | **bold**                    |
| `*italic*`    | Italic text | _italic_                    |
| `` `code` ``  | Inline code | `code`                      |
| `[link](url)` | Hyperlink   | [link](https://solidjs.com) |
