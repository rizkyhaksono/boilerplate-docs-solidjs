# Installation

This guide will walk you through setting up a new Natee Docs project from scratch.

## Prerequisites

Before you begin, make sure you have:

- **Node.js** version 22 or higher
- **bun** (recommended) or npm/yarn/pnpm

```bash
# Check your Node.js version
node --version

# Install bun (if not already installed)
curl -fsSL https://bun.sh/install | bash
```

## Create a New Project

### Option 1: Clone the Boilerplate

```bash
git clone https://github.com/rizkyhaksono/boilerplate-docs-solidjs my-docs
cd my-docs
bun install
```

### Option 2: Manual Setup

If you prefer to set things up manually:

```bash
# Create a new SolidStart project
npx -y create-solid@latest my-docs

# Navigate to the project
cd my-docs

# Install additional dependencies
bun add tailwindcss @tailwindcss/vite daisyui
bun add @mdx-js/rollup solid-jsx solid-mdx
bun add remark-gfm remark-frontmatter remark-mdx-frontmatter
bun add rehype-slug rehype-pretty-code shiki
```

## Development Server

Start the development server:

```bash
bun run dev
```

This will start the server at `http://localhost:3000` with hot module replacement enabled.

## Building for Production

Create a production build:

```bash
bun run build
```

Preview the production build:

```bash
bun run start
```

## Project Configuration

The main configuration files are:

| File                 | Purpose                  |
| -------------------- | ------------------------ |
| `app.config.ts`      | Vite & SolidStart config |
| `src/docs.config.ts` | Sidebar & site metadata  |
| `src/app.css`        | Global styles & Tailwind |
| `tsconfig.json`      | TypeScript configuration |
